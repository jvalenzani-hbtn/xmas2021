# Happy 2022!!

From: Holberton MVD Staff

## Instructions:

- Clone repository
- Edit tree.txt file and create your own tree.
  - `*` will be displayed in green
  - `$` will be displayed as random colored `0`
- Run `xmas.py` and enjoy. :)
